Growing Neural Cellular Automata

A Developmental-Biology Lab Bench

Grow a “creature” from a single seed cell and run classic embryology experiments — grafts, organizers, signalling beads, mutations, confinement, and regeneration.

An interactive teaching tool built on the neural cellular automata model. The seed is a zygote; the hidden channels stand in for morphogens (gene products); the network weights are the genome — a local “cell” program run everywhere at once. Everything runs in your browser; nothing is uploaded.

Open

Enter the Lab →

A demo organism loads automatically — just press Run and start experimenting. Cut and graft tissue, drop morphogen beads (including tissue-extract “cocktails”), watch every gene channel, plot gene expression over time, mutate the genome, and confine or damage growth. Advanced setup — loading your own model, grid size, dynamics, and the seed — lives under a collapsed “For experts” panel, so it’s there when you want it and out of the way when you don’t.

1 · Grow it

The demo organism is already loaded. Press Run to grow it from the seed, or Re-seed to start over.

2 · Experiment

Use the tools: lasso and cut tissue, graft pieces elsewhere, place signalling beads, and watch what develops.

3 · Measure

Open the gene atlas to see each morphogen, and lasso a region to plot its gene expression over developmental time.

4 · Go deeper

Open “For experts” to load your own trained model, change the grid, adjust the seed, or toggle developmental dynamics.

For teachers & developers: the demo organism is the file student_model.json served next to this page; replace it to change the default creature. Auto-load works when the page is served (e.g. GitHub Pages); if you open the file directly from disk, just expand “For experts” and load a model by hand. Models are produced by the accompanying PyTorch trainer. The bench fetches TensorFlow.js on first load, then runs entirely on your machine.
Built on “Growing Neural Cellular Automata” (Mordvintsev et al., Distill 2020). Teaching adaptation.
