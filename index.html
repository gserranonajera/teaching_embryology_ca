Growing Neural Cellular Automata

A Developmental-Biology Lab Bench

Grow a “creature” from a single seed cell and run classic embryology experiments — grafts, organizers, signalling beads, mutations, confinement, and regeneration.

These are interactive teaching tools built on the neural cellular automata model. The seed is a zygote; hidden channels stand in for gene expression; and the update rule is a local “cell” program applied everywhere at once. Everything runs in your browser — nothing is uploaded.

Tool 01

Lab bench →

Load a trained model and run it live. Cut and graft tissue, place morphogen beads, watch every gene channel in the atlas, confine growth, and toggle developmental dynamics.
Tool 02

Pattern painter →

Paint per-gene target maps — gradients, domains, and fields — and export them to guide training, so a chosen channel expresses in a pattern you design.
Getting started: open the lab bench and load a trained model file (a quantized JSON export or a full-precision graph model). If you don’t have one yet, the models are produced by the accompanying PyTorch trainer. The bench needs an internet connection the first time it loads (it fetches the TensorFlow.js library), then runs entirely on your machine.
Built on “Growing Neural Cellular Automata” (Mordvintsev et al., Distill 2020). Teaching adaptation.
